Detailed Description

Objects of this class represent UDP transport endpoints.

CHIP Inet Layer encapsulates methods for interacting with UDP transport endpoints (SOCK_DGRAM sockets on Linux and BSD-derived systems) or LwIP UDP protocol control blocks, as the system is configured accordingly.

Prepare the endpoint to receive UDP messages.

If mState is already State::kListening, then no operation is performed, otherwise the mState is set to State::kListening and the endpoint is prepared to received UDP messages, according to the semantics of the platform.

On LwIP, this method must not be called with the LwIP stack lock already acquired.

Parameters
[in]onMessageReceivedThe endpoint's message reception event handling function delegate.
[in]onReceiveErrorThe endpoint's receive error event handling function delegate.
[in]appStateApplication state pointer.
Return values
CHIP_NO_ERRORSuccess: endpoint ready to receive messages.
CHIP_ERROR_INCORRECT_STATEEndpoint is already listening.
