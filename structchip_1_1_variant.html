Detailed Description

template<typename... Ts>
struct chip::Variant< Ts >

Represents a type-safe union.

An instance of Variant at any given time either holds a value of one of its alternative types, or no value.

Example: struct Type1 {};

struct Type2 {};

Variant<Type1, Type2> v; v.Set<Type1>(); // v contains Type1 Type1 o1 = v.Get<Type1>();
