CHIP_ERROR chip::FabricTable::AddNewFabric ( FabricInfo &  fabric,
FabricIndex *  assignedIndex 
)

Add the new fabric information to fabric table if the table has space to store more fabrics.

CHIP_ERROR_NO_MEMORY error will be returned if the table is full.

The provided information will get copied to internal data structures, and the caller can release the memory associated with input parameter after the call is complete.

If the call is successful, the assigned fabric index is returned as output parameter.
