90 /*
91  * Overhead to encode a raw ECDSA signature in X9.62 format in ASN.1 DER
92  *
93  * Ecdsa-Sig-Value ::= SEQUENCE {
94  * r INTEGER,
95  * s INTEGER
96  * }
97  *
98  * --> SEQUENCE, universal constructed tag (0x30), length over 2 bytes, up to 255 (to support future larger sizes up to 512 bits)
99  * -> SEQ_OVERHEAD = 3 bytes
100  * --> INTEGER, universal primitive tag (0x02), length over 1 byte, one extra byte worst case
101  * over max for 0x00 when MSB is set.
102  * -> INT_OVERHEAD = 3 bytes
103  *
104  * There is 1 sequence of 2 integers. Overhead is SEQ_OVERHEAD + (2 * INT_OVERHEAD) = 3 + (2 * 3) = 9.
105  */
