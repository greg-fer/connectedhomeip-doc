◆ FreeRTOSTicksSinceBoot()

uint64_t chip::System::Clock::FreeRTOSTicksSinceBoot ( void  )

Returns the number of FreeRTOS ticks since the system booted.

NOTE: The default implementation of this function uses FreeRTOS's vTaskSetTimeOutState() function to get the total number of ticks, irrespective of tick counter overflows. Unfortunately, this function cannot be called in interrupt context, no equivalent ISR function exists, and FreeRTOS provides no portable way of determining whether a function is being called in an interrupt context. Adaptations that need to use the Chip Get/SetClock methods from within an interrupt handler must override this function with a suitable alternative that works on the target platform. The provided version is safe to call on ARM Cortex platforms with CMSIS libraries.
