This method cancels a one-shot timer, started earlier through StartTimer().

Note
The cancellation could fail silently in two different ways. If the timer specified by the combination of the callback function and application state object couldn't be found, cancellation could fail. If the timer has fired, but not yet removed from memory, cancellation could also fail.

◆ HandlePlatformTimer()

CHIP_ERROR chip::System::LayerImplLwIP::HandlePlatformTimer ( void  )

Handle the platform timer expiration event.

Completes any timers that have expired.

A static API that gets called when the platform timer expires. Any expired timers are completed and removed from the list of active timers in the layer object. If unexpired timers remain on completion, StartPlatformTimer will be called to restart the platform timer.

It is assumed that this API is called only while on the thread which owns the CHIP System Layer object.

Note
It's harmless if this API gets called and there are no expired timers.
Returns
CHIP_NO_ERROR on success, error code otherwise.

void *  aAppState 
)
overridevirtual

Schedules a function with a signature identical to OnCompleteFunct to be run as soon as possible in the CHIP context.

Note
This function could, in principle, be implemented as StartTimer. The specification for SystemTimer however permits certain optimizations that might make that implementation impossible. Specifically, SystemTimer API may only be called from the thread owning the particular System::Layer, whereas the ScheduleWork may be called from any thread. Additionally, whereas the SystemTimer API permits the invocation of the already expired handler in line, ScheduleWork guarantees that the handler function will be called only after the current CHIP event completes.
Parameters
[in]aCompleteA pointer to a callback function to be called when this timer fires.
[in]aAppStateA pointer to an application state object to be passed to the callback function as argument.
Return values
CHIP_ERROR_INCORRECT_STATEIf the System::Layer has not been initialized.
CHIP_ERROR_NO_MEMORYIf the SystemLayer cannot allocate a new timer.
CHIP_NO_ERROROn success.
