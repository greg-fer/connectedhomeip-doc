Transition from Uninitialized or Shutdown to Destroyed. 


Typical use is VerifyOrReturnError(state.Destroy(), CHIP_ERROR_INCORRECT_STATE); this function returns bool rather than a CHIP_ERROR so that error source tracking will record the call point rather than this function itself.


Returns
true if the state was Uninitialized or Shutdown and is now Destroyed. 



false otherwise.
