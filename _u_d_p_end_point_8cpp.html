Detailed Description

This file implements the Inet::UDPEndPoint class, where the CHIP Inet Layer encapsulates methods for interacting with UDP transport endpoints (SOCK_DGRAM sockets on Linux and BSD-derived systems) or LwIP UDP protocol control blocks, as the system is configured accordingly.
