CHIP_ERROR chip::JniReferences::GetClassRef ( JNIEnv *  env,
const char *  clsType,
jclass &  outCls

Creates a jclass reference to the given class type.

This must be called after SetJavaVm().

Parameters
[in]envThe JNIEnv for finding a Java class and creating a new Java reference.
[in]clsTypeThe fully-qualified Java class name to find, e.g. java/lang/IllegalStateException.
[out]outClsA Java reference to the class matching clsType.

◆ GetEnvForCurrentThread()

JNIEnv * chip::JniReferences::GetEnvForCurrentThread ( )

Returns a JNIEnv for the current thread.

This must be called after SetJavaVm(). If the current thread is not attached to the JVM, this method will attach the thread first, then retrieve the JNIEnv.
