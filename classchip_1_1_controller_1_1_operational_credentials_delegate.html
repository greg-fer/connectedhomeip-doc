This function generates an operational certificate chain for the device.

The API generates the certificate in X.509 DER format.

The delegate is expected to use the certificate authority whose certificate is returned in GetRootCACertificate() API call.

The delegate will call onCompletion when the NOC certificate chain is ready.

Parameters
[in]csrElementsCSR elements as per specifications section 11.22.5.6. NOCSR Elements.
[in]attestationSignatureAttestation signature as per specifications section 11.22.7.6. CSRResponse Command.
[in]DACDevice attestation certificate received from the device being commissioned
[in]PAIProduct Attestation Intermediate certificate
[in]PAAProduct Attestation Authority certificate
[in]onCompletionCallback handler to provide generated NOC chain to the caller of GenerateNOCChain()
Returns
CHIP_ERROR CHIP_NO_ERROR on success, or corresponding error code.
